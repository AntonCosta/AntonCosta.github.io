Website Title

Music Platformer

This is a very early version of what I hope to be a game one day. It is a fairly simple music platformer where every ability has a music note attached to it. The player's goal is to use the abilities in the correct order to play the song which is attached to the current level. I have also used this project to play a bit with Unity shaders and create a few visual effects. (The water in the image is not mine, it's from a Unity store package I've bought, only the ball smear and squish are mine.)
