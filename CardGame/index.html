Card Game

This project started as freelance work but at the moment it's on stand-by. I didn't give it's name but I can show you a bit of how it looks at the moment. It is a collectible card game, meant to be played online, with RPG elements and heavy emphasis on your hero card, the cards in your deck meant to support them.
The map is randomly generated, each hex providing different benefits and affecting each class differently.
The game is made in Unity using C# and I have used Mirror Networking for the multiplayer element. There are also a couple of shaders made by me using HLSL.
The server was deployed on AWS and the client as well. There is a matchmaking system in place and multiple games of two players can be held at the same time.
