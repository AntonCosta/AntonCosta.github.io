Footmen

This is an unreal engine project I've started a few weeks ago to learn the engine before UE5 Is released. I am working on it together with a friend and we hope to have a prototype fairly soon. The game is supposed to be a combination between Auto-Chess, a MOBA and Footmen Frenzy. It's is in extremely early stages; we are just leaning how to integrate C++ and the "Gameplay Abilities" plugin and we need to port what we had as blueprint behaviour into the new plugin.
